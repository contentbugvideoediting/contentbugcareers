# contentbug-careers

Content Bug careers page and multi-step application system.

## Stack
- Vanilla HTML/CSS/JS (zero dependencies)
- Airtable base: `appUvZqaCBZ42r9Nl` (Content Bug Careers)
- VideoTiny for video introductions
- Hosted on Railway or any static host

## Files
- `careers.html` - Full careers page + 3 wizard flows (Editor, PM, Sales)
- `AIRTABLE-SCHEMA.md` - Field reference

## Setup
1. Set `VIDEOTINY_ID` in the script config block of `careers.html`
2. Deploy `careers.html` as a static file
3. Airtable base is already live at `appUvZqaCBZ42r9Nl`
4. Font files reference `style-guide/fonts/` - copy from main portal repo or host separately

## Style
Uses the Content Bug brand style guide: Apple Liquid Glass, SF Pro fonts, CB color tokens.
